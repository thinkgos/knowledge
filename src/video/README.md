## SDP

`session description protocol`, 会话描述协议, 媒体协商

## STUN

`session traversal utilities for NAT`, NAT会话穿越应用程序, 网络信息交换, `RFC5839`

## TURN

`traversal using relays around NAT`, `RFC5839`扩展, 主要添加relay功能, 如果nat后, 与对等无法建立通信, 则需要公网服务器做中继

## ICE

`STUN`+`TURN`的一个信令服务器

## SSDP
